# `Bazinga (BZA)` edge-service
This application is part of the Polar Bookshop system that provides API gateway and cross-cutting concerns.

After building the application, you can also run it from the Java CLI:

```bash
$ java -jar target/order-service-0.0.1.jar
```

Run to build docker container locally

```bash
$ DOCKER_BUILDKIT=1 docker compose up -d
```
